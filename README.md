# SE-IR-A2
The	purpose	of	Assignment 2 is	to	learn	how	to	do	ranked	retrieval.	Tasks included	1)	how	
to	 include	 tf-idf	 scores	 in	 the	 inverted	 index;	 2)	 how	 to	 handle	 ranked	 retrieval	 from	
multiword	queries;	3)	how	to	use	PageRank	to	score	documents;	and	4)	how	to	combine	tfidf
and	PageRank	scoring.
